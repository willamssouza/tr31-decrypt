## 1 - Importing KEK

Importing KEK is a prerequisite for importing TR31 key blocks. The KEK is used to unwrap the TR31 key block and extract the underlying key material.

## 2 - Importing PEK

Importing PEK is a prerequisite for decrypting card data and translating PIN data. The PEK is used to decrypt the encrypted card data and PIN block.

## 3 - Importing TR31 Key Block of Card Data

Importing the TR31 key block of card data allows you to decrypt the encrypted card data and access the underlying information, such as the PAN.

## 4 - Decrypt Card Data

Decrypting card data allows you to convert the encrypted card data into its original form, making it readable and usable for further processing.

## 5 - Importing TR31 Key Block of PIN Data

Importing the TR31 key block of PIN data allows you to decrypt the encrypted PIN block and access the underlying PIN information.

## 6 - Translating PIN Data

Translating PIN data allows you to convert the encrypted PIN block from one format to another, such as from ISO Format 4 to ISO Format 0, while keeping the PIN information secure.
